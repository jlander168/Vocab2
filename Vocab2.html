Web Form Allows users to enter data that can be saved and processed
Controls/Widgets the objects that allow a user to interact with a form
Data Field stores data values supplied by a user
Input Box insert text and numeric values
option/radio Button select data values from a predefined set of options
Selection List select data values from an extensive list of options
Checkbox select data values limited to two possibilities, such as yes or no
Textarea Boxes enter text strings that may include several lines of content
Spin Box enter integer values confined to a specified range
Slider Controls enter numeric values confined to a specified range
Calendar Controls select data and time values
Color Picker choose color values
Get Method tells the browser to append the form data to the end of the URL specified in action
Post Method send the form data in its own separate data stream
