Web Form Allows users to enter data that can be saved and processed
Controls/Widgets the objects that allow a user to interact with a form
Data Field stores data values supplied by a user
Input Box insert text and numeric values
option/radio Button select data values from a predefined set of options
Selection List select data values from an extensive list of options
Checkbox select data values limited to two possibilities, such as yes or no
Textarea Boxes enter text strings that may include several lines of content
Spin Box enter integer values confined to a specified range
Slider Controls enter numeric values confined to a specified range
Calendar Controls select data and time values
Color Picker choose color values
Get Method tells the browser to append the form data to the end of the URL specified in action
Post Method send the form data in its own separate data stream
Fieldset groups fields that share a common purpose
Legend the content of a fieldset using < legend >
Virtual Keyboard software representations of a physical device
Spinner Control displays an up or down arrow to increase or decrease the field value by a set amount
Range Slider limits a numeric field to a range of possible values
Data Lists a list of possible data values that a form field can have
Form Buttons a type of form control that performs an action
Command Buttons runs a program that affects the content of a page or the actions of a browser
Submit Button submits a form to the server for processing when clicked
Reset Button resets a form to the server for processing when clicked
Validation process of ensuring that a user has supplied valid data
Server-side Validation validation occurs on a web server
Client-side Validation validation occurs in the user’s browser
Regular expression concise description of a character pattern
Inline Validation the technique of immediate data validation and reporting of errors
Focus the state in which an element has been clicked by the user, making it the active control on the form
